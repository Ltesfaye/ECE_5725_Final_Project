This code base is written to run on Python3 and 7" raspberyPI display.
- Before running main.py make sure you have the following libraries downloaded
	- Matplotlib
	- Numpy
	- bluedot
	- pybluez

- Also you will need to update your bluez following the tutrial bellow.
	https://learn.adafruit.com/install-bluez-on-the-raspberry-pi/installation

	testing
